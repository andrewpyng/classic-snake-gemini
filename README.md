# Classic Snake Game

This is a simple and classic implementation of the Snake game that runs directly in your browser.

## Features

- Classic snake gameplay
- Score and High Score tracking
- Selectable difficulty (Easy, Normal, Hard)
- Mobile-friendly with touch controls

## How to Play

1.  Open `index.html` in your web browser.
2.  Click "Start Game".
3.  Use the **Arrow Keys** on your keyboard to control the snake.
4.  On a mobile device, use the on-screen D-pad to control the snake.
5.  Eat the food to grow longer and increase your score.
6.  Avoid running into the walls or your own tail.

## Difficulty

You can select the difficulty before starting a new game. The options are:
- **Easy:** Slower pace.
- **Normal:** A balanced speed.
- **Hard:** A much faster and more challenging experience.

The game speeds up as your score increases, regardless of the difficulty setting.
